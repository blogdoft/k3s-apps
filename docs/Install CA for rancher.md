# Install CA

For some app, you will need to copy rootCA, generated by cert-manager, to namespace app scope.

Run this commands, updating to the proper namespace.

```bash
kubectl get secret -n cert-manager home-arpa-ca -o jsonpath='{.data.tls\.crt}' \
  | base64 -d > cacerts.pem

kubectl -n cattle-system create secret generic tls-ca \
  --from-file=cacerts.pem \
  --dry-run=client -o yaml > rancher-tls-ca.yaml  

kubectl apply -f rancher-tls-ca.yaml

rm rancher-tls-ca.yaml
rm cacerts.pem
```
